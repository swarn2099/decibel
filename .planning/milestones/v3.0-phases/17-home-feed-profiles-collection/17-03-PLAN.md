Web API endpoints used (call these from mobile, NOT direct Supabase — they handle tier calculation and founder badge logic):

POST https://decibel-three.vercel.app/api/collect
Body: { performer_id: string, email: string }
Response: { scan_count: number, current_tier: string, already_collected: boolean }

POST https://decibel-three.vercel.app/api/discover
Body: { performer_id: string }
Headers: Authorization via Supabase session token
Response: { success: boolean, collection_id?: string, performer_id?: string, performer_name?: string, performer_slug?: string, already_discovered?: boolean, is_founder?: boolean }

NOTE on /api/discover auth: The web discover endpoint uses createSupabaseServer() to get the user from cookies. Since mobile doesn't have cookies, the mobile app needs to send the Supabase access token. The discover endpoint will need to accept a bearer token OR the mobile can call Supabase directly for the discover flow. Use the direct Supabase approach:
1. Get user session from auth store
2. Insert into collections table (fan_id, performer_id, capture_method='online', verified=false)
3. Try insert into founder_badges (unique constraint handles duplicates)
4. Upsert fan_tiers

For /api/collect, the endpoint uses email (no auth cookies required), so mobile can call it directly with the user's email.

Tier system:
- network: 1+ scans (color: pink #FF4D6A)
- early_access: 3+ scans (color: purple #9B6DFF)
- secret: 5+ scans (color: blue #4D9AFF)
- inner_circle: 10+ scans (color: teal #00D4AA)

Shareable card generation:
- POST https://decibel-three.vercel.app/api/social/collection-card
- Returns image URL that can be shared

From PRD — Confirmation animation spec:
- Full-screen takeover with backdrop blur
- Stamp press animation: artist photo animates in like stamp being pressed — starts above, slams down with haptic thud, slight bounce, ink ring expands
- Tier badge seals into corner like hot wax
- "Collected!" text with celebration
- If tier-up: wax seal cracks and reforms in new tier color with confetti
- "Share" and "Done" buttons at bottom
- Auto-dismiss after 5 seconds if no interaction

Haptics:
- Medium impact on collect/discover (expo-haptics ImpactFeedbackStyle.Medium)
- Heavy impact on tier-up (expo-haptics ImpactFeedbackStyle.Heavy)
