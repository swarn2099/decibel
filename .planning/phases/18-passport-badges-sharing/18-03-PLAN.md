  <action>
Install required packages if not present: `expo-sharing`, `expo-file-system`, `expo-clipboard` (check package.json first — some may already be installed from Phase 16/17).

Create `src/hooks/useShareCard.ts`:
- `usePassportShareCard()` — returns async function that:
  1. Takes PassportStats + fan name + slug
  2. Builds URL to `https://decibel-three.vercel.app/api/passport/share-card` with query params
  3. Downloads image to temp file via expo-file-system (FileSystem.downloadAsync to cacheDirectory)
  4. Returns local file URI for sharing

- `useArtistShareCard()` — returns async function that:
  1. Takes artist name, photo_url, tier, scans, venue, fan name, slug
  2. Builds URL to `.../share-card/artist` with query params
  3. Downloads to temp file, returns URI

- `useBadgeShareCard()` — returns async function that:
  1. Takes badge_id, fan name, icon, rarity, description
  2. Builds URL to `.../share-card/badge` with query params
  3. Downloads to temp file, returns URI

- `useSharePassportLink()` — returns async function that:
  1. Takes fan slug/username
  2. Copies `https://decibel-three.vercel.app/u/{slug}` to clipboard via expo-clipboard
  3. Returns the URL string

Each hook returns `{ generate, isLoading, error }` pattern using useState internally.

Create `src/components/passport/ShareSheet.tsx`:
- Bottom sheet modal (use React Native Modal with slide-up animation from bottom)
- Dark background (#15151C), rounded top corners
- Title: "Share" in Poppins Bold
- Preview: show the share card image (passed as URI prop) as a small preview (200px tall)
- Options as horizontal icon row:
  1. Instagram Stories — icon, "Stories" label. Uses Sharing.shareAsync with `{ mimeType: 'image/png', UTI: 'com.instagram.stories' }`. If Instagram not installed, fall back to general share.
  2. iMessage — icon, "Message" label. Uses Share.share with message + URL.
  3. Copy Link — icon, "Copy Link" label. Copies the deep link URL to clipboard, shows "Copied!" toast (brief text change).
  4. Save — icon, "Save" label. Uses MediaLibrary.saveToLibraryAsync (need expo-media-library). Request permissions if needed.
- Loading spinner while card is being generated
- Medium haptic impact on share action (expo-haptics, ImpactFeedbackStyle.Medium)
- Close handle bar at top (small gray bar, drag indicator)

Create `src/components/passport/PassportShareButton.tsx`:
- Gradient button: purple (#9B6DFF) to pink (#FF4D6A) using LinearGradient from expo-linear-gradient
- "Share Passport" text in white Poppins Bold with arrow-up-right icon (from lucide-react-native)
- Full width, rounded-xl, padding 16px vertical
- On press: calls the share flow (passed as onPress prop)
