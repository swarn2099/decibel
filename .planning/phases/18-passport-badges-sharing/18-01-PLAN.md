Support infinite scroll via TanStack useInfiniteQuery with 20 items per page.

3. `useArtistTierProgress(performerId)` — Query fan_tiers for specific fan+performer pair. Return { scanCount, currentTier, nextTier, scansNeeded }. Tiers: network=1, early_access=3, secret=5, inner_circle=10.

For seeded rotation: Use a simple hash of collection id (sum of charCodes mod 7 - 3 gives -3 to +3 degrees). This ensures stamps stay in the same position on re-renders.
