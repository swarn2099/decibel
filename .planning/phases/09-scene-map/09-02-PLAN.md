<task type="auto">
  <name>Task 1: Install Leaflet and build the map page</name>
  <files>src/app/map/page.tsx, src/app/map/map-client.tsx, src/app/globals.css</files>
  <action>
1. Install dependencies: `npm install leaflet react-leaflet` and `npm install -D @types/leaflet`

2. Create `src/app/map/page.tsx` — thin server component:
   - Metadata: title "Scene Map | DECIBEL", description "Explore Chicago's underground music scene"
   - Renders MapClient component
   - No data fetching here (client component handles it for interactivity)

3. Create `src/app/map/map-client.tsx` — "use client" component:

   **Map setup:**
   - Use react-leaflet's MapContainer, TileLayer, CircleMarker, Popup
   - Dark tile layer: `https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png` (free, dark theme, no API key)
   - Center: [41.8827, -87.6233] (Chicago loop), zoom: 12
   - Import Leaflet CSS via `import "leaflet/dist/leaflet.css"`
   - Dynamic import the map component with `next/dynamic` and `ssr: false` (Leaflet requires window)

   **Venue markers:**
   - Render each venue as a CircleMarker (not a default Leaflet marker — circle dots look better on dark maps)
   - Dot color: var(--pink) (#FF4D6A) for venues with events, slightly transparent
   - Dot radius: scale by event_count — min 6px (1 event) to max 14px (10+ events). Formula: `Math.min(6 + venue.event_count * 1.5, 14)`

   **Venue popups:**
   - On click, show Leaflet Popup styled to match Decibel dark theme
   - Content: venue name (bold, white), then list of upcoming events (max 5):
     - Each event row: performer name (linked to /artist/{slug}), date formatted as "Mon DD", time if available
   - Show "top performers" as small circular avatars (performer_photo) if available, max 3
   - Style the popup with dark bg (#15151C), light text, pink accent for links
   - Use Leaflet's popup className option to apply custom styles

   **Genre filter:**
   - Above the map, render a horizontal scrollable pill bar of genre options
   - Extract unique genres from all venue data on initial load
   - Show top 10 most common genres as pills, plus an "All" option (selected by default)
   - When a genre pill is tapped, re-fetch from /api/map?genre={selected} and update markers
   - Active pill: solid pink background. Inactive: bg-card border with gray text
   - Pills should be horizontally scrollable on mobile (overflow-x-auto, no-scrollbar)

   **Tonight mode:**
   - Toggle button in the filter bar: "Tonight" with a pulsing dot icon
   - When active, re-fetch from /api/map?tonight=true
   - Active venue dots get a CSS pulsing animation (scale 1 -> 1.4 -> 1, opacity pulse)
   - Tonight button: when active, uses pink bg with white text; inactive: bg-card with gray text
   - Genre filter and Tonight can be combined: /api/map?genre=techno&tonight=true

   **Mobile responsive (MAP-06):**
   - Map takes full viewport height minus top bar: `h-[calc(100vh-120px)]` or similar
   - Filter bar stacks above map, horizontally scrollable
   - Popups: maxWidth 280px so they don't overflow on mobile
   - Touch-friendly: CircleMarker radius minimum 6px for tap targets
   - No horizontal scroll on the page itself

   **Loading state:**
   - Show a skeleton/spinner while fetching map data
   - Use the existing motion library for fade-in when map loads

4. Add to `src/app/globals.css` — pulsing dot animation:
   ```css
   @keyframes pulse-dot {
     0%, 100% { transform: scale(1); opacity: 0.9; }
     50% { transform: scale(1.4); opacity: 0.6; }
   }
   .pulse-dot {
     animation: pulse-dot 2s ease-in-out infinite;
   }
   ```

   Also add Leaflet popup dark theme overrides:
   ```css
   .leaflet-popup-content-wrapper {
     background: #15151C !important;
     color: #ededed !important;
     border-radius: 12px !important;
     border: 1px solid rgba(110, 110, 130, 0.3) !important;
   }
   .leaflet-popup-tip {
     background: #15151C !important;
   }
   .leaflet-popup-close-button {
     color: #A0A0B0 !important;
   }
   ```
  </action>
