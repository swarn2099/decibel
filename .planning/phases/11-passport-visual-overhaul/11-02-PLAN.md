Create `src/lib/fan-slug.ts` with:
```typescript
export function generateFanSlug(fan: { name: string | null; id: string }): string {
  if (fan.name) {
    return fan.name.toLowerCase().replace(/[^a-z0-9]+/g, '-').replace(/^-|-$/g, '') || fan.id.slice(0, 8);
  }
  return fan.id.slice(0, 8);
}
```

**B. Create src/app/api/og/passport/route.tsx (OG image generation):**

Use Next.js built-in `ImageResponse` from `next/og` (available in Next.js 15, no extra dependency needed).

Edge runtime route that generates a 1200x630 OG preview image:

```typescript
import { ImageResponse } from 'next/og';

export const runtime = 'edge';

export async function GET(request: Request) {
  const { searchParams } = new URL(request.url);
  const name = searchParams.get('name') || 'Fan';
  const artists = searchParams.get('artists') || '0';
  const shows = searchParams.get('shows') || '0';
  const venues = searchParams.get('venues') || '0';
  const genre = searchParams.get('genre') || '';

  return new ImageResponse(
    // JSX layout — dark background, Decibel branding
    // Top: DECIBEL logo text (gradient)
    // Center: "[Name]'s Passport" in large bold text
    // Bottom row: stat pills — "X Artists" "Y Shows" "Z Venues"
    // Optional: favorite genre tag
    // Colors: bg #0B0B0F, pink #FF4D6A, purple #9B6DFF, blue #4D9AFF, teal #00D4AA
    // Font: system default (satori doesn't support custom fonts easily on Edge without embedding)
    // Keep it clean, intriguing, drives clicks
    ,
    { width: 1200, height: 630 }
  );
}
```

Design the OG card to be:
- Dark background (#0B0B0F)
- "DECIBEL" at top in gradient text
- Fan name centered, large
- Stats row at bottom with colored numbers
- Subtle gradient border or accent line
- The vibe should make someone want to click through

IMPORTANT: `ImageResponse` uses satori which only supports a subset of CSS (flexbox, no grid). Use `display: 'flex'` everywhere. Style objects, not className.
