   - Left section (~40%): "DECIBEL" in bold white text (36px), fan name (28px), "{N} artists collected" (20px, #A0A0B0), tier breakdown summary (e.g., "2 Inner Circle, 3 Secret")

IMPORTANT: Satori (used by ImageResponse) only supports flexbox layout with inline `style` props. No CSS classes, no Tailwind, no grid. Use `display: 'flex'`, `flexDirection`, `flexWrap`, `alignItems`, `justifyContent`.
