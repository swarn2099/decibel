      rank: number; fanId: string; name: string;

      rank: number; performerId: string; name: string;
      slug: string; photoUrl: string | null; fanCount: number;

    2. Create src/hooks/useLeaderboard.ts:
    - Export `useLeaderboard({ tab, period }: { tab: LeaderboardTab; period: TimePeriod })`
    - Uses TanStack Query with key `["leaderboard", tab, period]`
    - **Fan leaderboard query** (mirrors web page.tsx logic):
      a. Determine date filter: weekly = 7 days ago, monthly = 30 days ago, allTime = no filter
      b. Query `collections` table: `supabase.from("collections").select("fan_id, fans!inner(id, display_name, email)")`
      c. If not allTime, add `.gte("collected_at", dateFilter)`
      d. Group by fan_id client-side: count collections per fan, derive topTier from count using thresholds (1=network, 3=early_access, 5=secret, 10=inner_circle)
      e. Sort by count desc, assign rank 1..N
      f. Use display_name if available, otherwise derive from email (before @), NEVER expose full email
      g. Return top 100 as FanLeaderboardEntry[]
    - **Performer leaderboard query**:
      a. Query `collections` table grouped by performer_id, join performers for name/slug/photo/genres
      b. Same date filter logic
      c. Count unique fan_ids per performer = fanCount
      d. Sort by fanCount desc, assign rank
      e. Return top 100 as PerformerLeaderboardEntry[]
    - Also return `currentFanId` from authStore for highlighting
    - staleTime: 5 minutes
    - Also export a helper `getTierColor(tier: string): string` that maps tier names to hex colors: network=#FF4D6A, early_access=#9B6DFF, secret=#4D9AFF, inner_circle=#00D4AA

    3. Create src/components/leaderboard/RankRow.tsx:
    - Export `FanRankRow` component:
      - Props: `entry: FanLeaderboardEntry, isYou: boolean`
      - Layout: rank number (bold, left), initials circle (40px, bg card), name (truncated), count ("X collected"), tier badge pill
      - If isYou: teal left border (3px), subtle teal glow background (rgba(0,212,170,0.08))
      - Rank 1-3 get special colors: #1=pink, #2=purple, #3=blue

    - Export `PerformerRankRow` component:
